🎨 1. APP VIBE OVERVIEW
Core Feel

Tone: Energetic, competitive, social, and clean — a blend of trading terminal + mobile game.

Color Palette:

Deep charcoal / black background (#0B0B0E)

Neon gradients for highlights (turquoise #2BFBD2, magenta #E652F8)

Muted gray text for non-active states

Gold glow for victory / Alpha Rank highlights

Typography

Title font: Orbitron / Space Grotesk (tech, futuristic)

Body font: Inter / Poppins for readability

Numbers / scores: JetBrains Mono (gives quant + gamer vibe)

Iconography

Minimal line icons (Lucide / Remix)

Animated reaction icons for result reveal (🔥, 🧠, 🥶)

UX Principles

One core decision per screen (no clutter).

Keep tension: hide crowd results until reveal.

Emphasize feedback loops — motion, streaks, sound.

Always show countdowns — users should feel the ticking clock.

Purpose: Daily question at the center of attention.

Layout

🧩 3. GAME MECHANICS SUMMARY
Feature	Description	Implementation Hint
Hidden Sentiment	Hide crowd % until window closes	if now < close_time → hide chart
Daily Drop	New question daily @ 12 PM ET	Cron + status=‘scheduled’
Voting Limit	5 votes per day	user.votes_used_today < 5
Public/Private Mode	Toggle on vote card	is_public boolean
Scoring	Public: 100 pts × rarity multiplier; Private: 50 pts × rarity multiplier	SQL trigger / cron
Reveal	12 hours later	Auto job + animation
Leaderboards	Weekly + All-Time	SQL aggregate by points
Badge System	Milestones (streaks, accuracy, transparency)	Secondary table or metadata
