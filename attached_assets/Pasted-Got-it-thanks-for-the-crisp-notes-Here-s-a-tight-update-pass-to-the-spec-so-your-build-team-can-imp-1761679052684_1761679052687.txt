# Daily Question App — Spec Update (v2)

## Core changes

1. **Remove vote caps**

   * **Delete**: “5/5 votes used” meter everywhere.
   * **No throttling**: Users can answer the daily question once per account; no separate “votes” concept.

2. **“Next drop” → “Next question”**

   * Replace all UI text: **Next Drop** → **Next Question**.

3. **Daily reset**

   * **Reset time**: Every day at **12:00 p.m. ET** (11:00 a.m. CT).
   * **Countdown**: Show `HH:MM:SS until next question` wherever the question card appears.
   * **Rule**: One answer per user per daily question; lock answers after submission until the next reset.

4. **Profile & Leaderboard access**

   * Add a top-right **Profile** icon/button.

     * **Profile screen** shows:

       * Username (editable if allowed by policy).
       * Points balance.
       * Streak (consecutive days answered).
       * History: list of past questions, user’s answer, outcome (Majority/Public/Private), points earned.
     * Actions: **Edit username** (if enabled), **View history** (deep link to the list).
   * Add a **Leaderboard** button on the Home and Profile.

     * Tabs: **Daily**, **Weekly**, **All-time**.
     * Each row: rank, username, points, streak badge.

5. **Public vs Private toggle (points multiplier)**

   * Toggle lives on the answer composer.
   * **Public = ×2 points**, **Private = ×1 point**.
   * On toggle, show a live multiplier pill next to the submit button:

     * Public: `×2 Points` (use a bold accent).
     * Private: `×1 Point` (neutral tone).
   * Also show a tiny helper text:

     * Public: “Your answer will appear on your profile & public feeds.”
     * Private: “Only you can see this in History.”

6. **Lightning bolt icon ⚡**

   * Purpose: **Points** shorthand + **streak power** indicator.
   * Use it consistently as the **points glyph**:

     * Next to the points balance in the header: `⚡ 1,240`.
     * On results: `+⚡20` (Public) or `+⚡10` (Private).
     * On streak badge: a small ⚡ overlay to signal streak multiplier (e.g., `Streak ×1.1`).
   * If you **don’t** want streaks/multipliers beyond the Public/Private rule, keep ⚡ purely as the **points icon** (no extra logic).

## Logic rules (succinct)

* **Reset**: `T_reset = 12:00:00 ET` daily. Countdown = `T_reset – now`.
* **Visibility multiplier**:

  * Public: `award = base_points × 2`
  * Private: `award = base_points × 1`
* **Results privacy**: No aggregate sentiment, no majority % until after reveal.
* **One answer/day**: Enforce per user ID.
* **Username**: unique validation (if editable). Rate-limit changes.

---

## Edge cases

* User opens just before reset (<10s): submitting after cutover applies to **new** day; show toast if cutoff occurs during submit.
* Timezone display: lock all reveal messaging to **ET** and optionally show “(resets 11:00 a.m. your time)” for Central users.
* Offline: allow drafting; submission requires online; cache last seen countdown.
